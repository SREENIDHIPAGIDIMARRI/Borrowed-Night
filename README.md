# Borrowed Night

## Concept
This project visualizes the relationship between teenage screen time and sleep loss using a metaphorical AI-generated image.

## Insight
Teenagers are losing sleep not because the night got shorter, but because screens are taking its place.

## Visual Metaphor
A teenager is shown lying in bed under a glowing smartphone screen instead of a blanket, symbolizing digital devices replacing rest.

## Why This Works
The image avoids standard chart forms and communicates both the fact and feeling of the data insight without requiring labels or explanatory text.
